How It Works

Three layers of defense

Aegis intercepts the path between intent and execution, catching threats at every stage.

01

Connect

Add Aegis to your agent with a single command. Works with Claude, GPT, and any MCP-compatible client. Your agent gains four safety tools instantly.

02

Scan & Simulate

Before any trade, Aegis scans the target contract against 22 exploit patterns and simulates the transaction on a forked chain. No real funds at risk.

03

Enforce

On-chain contracts provide a final safety net. AegisGateway routes transactions through verified checks. The Uniswap v4 hook blocks unsafe swaps at the pool level.

Integration

Running in under 30 seconds

One command for Claude Code. One JSON block for any MCP client.

Terminal 
# Add Aegis to Claude Code
$ claude mcp add aegis npx aegis-defi

# Or run directly
$ npx aegis-defi

# Your agent now has 4 safety tools:
#   scan_contract      - Static analysis for 22 exploit patterns
#   simulate_transaction - Forked-chain tx simulation
#   check_token        - Anti-honeypot token checks
#   assess_risk        - All-in-one risk assessment
JSON - claude_desktop_config.json 
{
  "mcpServers": {
    "aegis": {
      "command": "npx",
      "args": ["aegis-defi"]
    }
  }
}
TypeScript 
import { createAegisServer } from "aegis-defi";

// Create and start the MCP server
const server = createAegisServer();

// Connect via stdio transport
const transport = new StdioServerTransport();
await server.connect(transport);

// The server exposes 4 tools:
//   scan_contract, simulate_transaction,
//   check_token, assess_risk
// assess_risk returns signed attestations for
// ALLOW/WARN decisions (chainId-bound signatures)
On-Chain Enforcement

Safety at the protocol level

Smart contracts on Base provide a final, trustless safety net that no off-chain tool can bypass.

AegisGateway Base

Routes agent transactions through on-chain safety checks. Collects a 5 bps fee per transaction. Safe multisig is set at deployment - no admin key can redirect funds. Signatures include chain ID + contract address to prevent cross-chain replay.

Network Base (Chain ID 8453)
Fee 5 bps (0.05%), nonReentrant
Fee Wallet Safe multisig, immutable
Security EIP-2 check, ecrecover guard, rescueStuckEth

AegisHook Uniswap v4

A Uniswap v4 hook that runs safety checks before any swap executes. Operates at the pool level for automatic protection - unsafe swaps are blocked before they can settle. Hook owner is immutable. Deployed via CREATE2 at vanity address.

Type beforeSwap hook, immutable owner
Scope Pool-level enforcement
Action Blocks flagged token swaps
Events RiskThresholdUpdated, PermissiveModeUpdated, AttestationRecorded

How fees work

Transparent on-chain fee

AegisGateway collects 0.05% on transactions routed through the gateway. This is enforced in the smart contract and visible to all parties.

Safe multisig fee recipient

Fees are sent to a Safe multisig set at deployment time. No admin function can change the destination. withdrawFees is protected by nonReentrant. No rug-pull vector.

MCP tools are free

Off-chain scanning, simulation, and risk assessment carry no fee. Only on-chain gateway routing has a cost.
